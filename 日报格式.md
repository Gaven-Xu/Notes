# 助教日报格式：

```
1、出勤：当天出勤、请假人数。姓名+请假原因。迟到早退人数及原因
2、课堂纪律：违反纪律名单，存在问题，解决办法
3、授课反馈：学员对当天授课过程中所存在的疑问汇总，是否及时通知授课讲师。是否给予相应的解决办法
4、教学环境：教室环境（网络、空调、门、窗）是否存在问题，是否已经通知行政部进行处理
5、考试情况：每天测评、周考、阶段考试所发现的问题，末位关怀学生名单（存在的问题、学习态度、接受能力等）
6、视频上传：当天视频是否已经全部上传
```

# 讲师日报格式：

```
1、教学环境：同步班级是否能够及时上线，各地网络是否存在问题。声音传输是否有问题，早、中上课正式讲课时间
2、课堂纪律：学生当天听课的表现，是否存在开小差、玩手机、看视频等情况，是否给予指正。违纪学生名单
3、授课内容：当天授课重点。通过授课学生集中表现出来的问题，解决办法。下课前是否有对当天课程回顾总结
4、考试情况：对每天测评、周考、阶段考试进行汇总，解决学生通过考试集中反应出的问题进行解答
5、视频资料：是否将当天录制视频传给助教，第二天的授课资料是否提前发到QQ讨论组中
6、备课：当天备课准备到哪个阶段什么内容，知识点是什么？需要提供当天备课资料附件照片。（针对没有授课情况，备课的讲师）
```
