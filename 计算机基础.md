# 网络原理
===
域名（网址）-> (DNS) -> 外网IP地址
> IP 能够找到某一个局域网

外网IP -> (路由器) -> 内网IP 、 MAC地址
> MAC地址，可以找到指定的电脑、网卡

> 端口 能够找到电脑上指定的软件

> 协议，规定了发送数据的格式和方式

http://www.baicu.com:80;

http 协议
www 主机
baidu.com 网址
80 端口
